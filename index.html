Library Management System – C++ with SQLite-ORM
Collaborated with two teammates to develop a robust console-based Library Management System using C++ and the SQLite ORM library. The system features dynamic database schema creation with entity relationships between authors, books, borrowers, and borrowing records—ensuring data consistency through foreign key constraints, cascade delete, and update restrictions. We implemented both persistent and in-memory database modes to support real-world usage and unit testing. Core functionalities include managing authors, books, and borrowers; tracking borrowing and return activities; and generating clean, structured listings with real-time status updates. The interface was carefully designed to offer intuitive menu navigation, interactive prompts, and helpful error handling, all focused on providing a smooth user experience. This project reflects our strong collaboration, clean architecture, and passion for building scalable and maintainable systems.

Snake Game – C++ with SFML 2
Developed a modern and engaging twist on the classic Snake game using C++ and the SFML 2 graphics library. This solo project focuses on creating a dynamic and replayable experience through random map generation, which produces new obstacle layouts each run and stores them via file I/O for persistence. The game introduces a coin collection system tied to a functional in-game store, allowing players to shrink the snake, buy extra lives, or slow down the game speed—adding layers of strategy and fun. Built with CLion, the game balances traditional mechanics with enhanced gameplay, encouraging both quick reflexes and tactical decision-making. While fully playable, the game remains an evolving project, with future improvements planned. This project showcases creativity, problem-solving, and a strong grasp of interactive game development in C++.
